## App for people who want to use less plastic :whale:
Server repo of full-stack app for searching all plastic free products in your city

## Technologies used

- Node.js, Express.js
- REST API
- JSON Web Tokens (JWT), BCrypt
- PostgreSQL, Sequelize ORM, Docker
- Google Places API

## Setup

In order to run this App:
- git clone git@github.com:viktoriachernykh/plastic-free-server.git
- npm install
- ensure you have Google API key set as environment variable (API_KEY="your key here")
- nodemon
